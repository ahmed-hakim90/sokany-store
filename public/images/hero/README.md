# صور بانر الهوم (Hero)

أي صورة تحطها هنا تظهر تلقائيًا في بانر الصفحة الرئيسية.
لا حاجة لتعديل أي كود — التحميل يتم من الخادم عبر
`features/home/services/getHeroSlides.ts`.

## المواصفات الموصى بها
- المقاس: 330x540 (أو مضاعفاتها مثل 660x1080 لشاشات Retina)
- الصيغة: WebP أو JPG (مدعوم: webp / avif / jpg / jpeg / png / gif)
- الجودة: 80-90%

---

## الوضع الأول: تلقائي (بدون manifest)
- ضع أي عدد من الصور هنا.
- تُعرض كلها بترتيب أبجدي (مع دعم الترقيم الطبيعي).
- للتحكم في الترتيب استخدم بادئة رقمية:
  - `01-promo.webp`
  - `02-summer.jpg`
  - `03-winter.png`
- الـ `alt` يُولَّد تلقائيًا من اسم الملف.
- لا توجد روابط (`href`) عند الضغط.

---

## الوضع الثاني: تحكم كامل (manifest.json)
أنشئ ملف `manifest.json` في نفس الفولدر للتحكم في:
- ترتيب الصور
- نص `alt` بالعربي
- الرابط (`href`) عند الضغط على كل صورة
- إخفاء صورة (لا تذكرها في الـ manifest)

### مثال جاهز
انسخ `manifest.example.json` إلى `manifest.json` ثم عدّله:

```json
[
  {
    "file": "01-home-appliances.webp",
    "alt": "أجهزة منزلية من سوكاني",
    "href": "/products"
  },
  {
    "file": "02-kitchen.webp",
    "alt": "عروض المطبخ",
    "href": "/categories"
  }
]
```

### قواعد سلوك الـ manifest
- لما يكون موجود → تُعرض **فقط** الملفات المذكورة فيه، بنفس الترتيب.
- أي ملف مذكور في الـ manifest لكن غير موجود فعليًا → يُتجاهل.
- لو الـ manifest فاضي أو غير صالح → يرجع للوضع التلقائي.

---

## ملاحظات عامة
- الملفات اللي تبدأ بنقطة (`.`) يتم تجاهلها.
- لو الفولدر فاضي، البانر مايتعرضش على الصفحة الرئيسية.
- الـ `manifest.example.json` للتوثيق فقط ولا يؤثر على العرض.
- بعد إضافة أو استبدال ملفات JPG كبيرة، شغّل من جذر المشروع: `npm run optimize:hero-images` (يضبط الحد الأقصى 660×1080 داخل الإطار ويقلّل الحجم).

## الهيرو من Firestore (لوحة التحكم)
إذا كان الموقع يعرض شرائح من مستند `home_hero` في Firestore بدل الملفات هنا، فالصور المرفوعة إلى التخزين (مثل Firebase Storage) يجب أن تلتزم بنفس الانضباط: عرض تقريبي **660px** كحد أقصى للمصدر، بدون PNG شفافة ضخمة، حتى يبقى LCP على الموبايل خفيفًا. المتصفح ما زال يمرّر الروابط عبر `next/image` (AVIF/WebP حسب الدعم).
